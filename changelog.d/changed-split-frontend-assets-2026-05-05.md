- Split the bulk CSS and JS out of `static/index.html` into `static/app.css` and `static/app.js`. The HTML drops from ~18k lines to ~1.5k. No bundler, no npm — plain `<link>` and `<script src>`. Small bootstrap blocks (FOIT theme guard, Morning feature-flag loader, loading-overlay renderer, in-UI terminal, history-search drawer) stay inline because they need to run before the main app boots. Server gains a generic `/static/` handler restricted to `.css`/`.js` with the same path-traversal guard as the existing Morning handler.
