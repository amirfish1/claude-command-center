Two bets nobody else makes

Attach, don't own.

Most tools only see sessions they launched. CCC reads your agent's on-disk state as the source of truth, so a session you started in any terminal, by any tool, shows up automatically. Nothing slips through.

Issues are the state machine.

Start from a GitHub issue in one click. CCC auto-labels it in-progress, renders the issue and comments on the card, and closes it with the commit SHA when you verify. A full bidirectional loop, not just a list.

What's on the board

Built for shipping many things at once.

Flow canvas

A Miro-style spatial board of repos → objects → sessions. Arrange your work in space, not just a list.

Signal-driven kanban

Eight columns, auto-classified from each session's real state. Drag to override; overrides decay back to the truth.

Group chat

N sessions coordinate through a shared file — agents talk to each other while they work.

Multi-engine

Claude Code, Codex, Cursor, and Antigravity on one board. Spawn across all of them.

Auto-fix-deploy

Polls your deploys; on a new production error it spawns a fix session, deduped by SHA. Nobody else ships this.

Resume on demand

Wake a dormant session and keep talking to it from the browser — the input pipe stays open.
