# Public copy guidance

The detailed CCC messaging source is maintained outside this public source
tree. This file remains as a public-safe anchor for links from the proof
inventory.

## 5. Problem families

Public claims must be specific, truthful, and supported by a **Built** entry in
[`pain-feature-proof.md`](pain-feature-proof.md). The public site and updates
should describe shipped behavior, not private campaign strategy or drafts.
