The rail fires at focus — it needs only the session row, never your text. The route stack re-ranks on intent: question-shaped text promotes Search, task-shaped text promotes Continue. Nothing here is labelled Send.

The launch spec is stated, not chosen — engine, model, and effort, the same triple the queue config uses. Defaults are opinionated: same engine as the origin for tool continuity, Sonnet because a scoped continuation is real work but not Opus work, Light effort because the hard thinking already happened upstream. Switching engine re-scopes the model list. Routes that run no model show no launch line at all.
